## ğŸ› Troubleshooting

### If images don't generate:

1. **Check API keys are saved**:
   - Go back to Settings
   - Keys should be there (they're write-only, so you won't see them)
   - Try saving again

2. **Check backend logs**:
   ```bash
   # Look for errors like:
   ERROR - Stable Horde error: ...
   WARNING - Hugging Face model failed: ...
   ```

3. **Check browser console**:
   - Any red errors?
   - Is polling happening? (requests every 5s)

4. **Try generating a new story**:
   - Don't retry old stories
   - Generate a fresh one with images enabled

### If it says "No providers configured":
- Your API keys weren't saved properly
- Go to Settings and save them again
- Make sure you clicked "Save Image Keys" button

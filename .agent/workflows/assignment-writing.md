# Writing Assignment Workflow

## Phase 1: Configuration & Curriculum Link
**Goal:** Define the writing task.
1.  **Learning Path**: Link to relevant Node (e.g., "Unit 4: Expressing Opinions").
2.  **Settings**:
    - **Min Word Count**: e.g., 200 words.
    - **Time Limit**: Optional (e.g., 30 mins).
    - **Grading Mode**: "Teacher Only", "AI Assisted", or "AI Auto-Grade".

## Phase 2: Content Creation
**Goal:** Set the prompt.
1.  **Prompt Builder**:
    - **Topic**: "Should public transport be free?"
    - **Guidance Params**: "Use passive voice twice", "Include 3 specific vocab words".
    - **Rubric**: Define criteria (Grammar, Coherence, Vocabulary).
2.  **AI Assist**: "Generate a B1 level writing prompt about 'Environment'".

## Phase 3: Student Execution (Writing Player)
**Goal:** Production.
1.  **Clean Interface**: Distraction-free text editor.
2.  **Tools**:
    - Word counter.
    - Spell-check (optional toggle).
    - "Stuck?" Button (AI provides a generic starter sentence, penalized score).
3.  **Submission**: Autosave & Final Submit.

## Phase 4: Review & Grading (The Grader)
**Goal:** Feedback.
1.  **AI Evaluation (Agent)**:
    - Runs first (if enabled). Checks grammar, coherence, rubric usage.
    - Generates *suggested* score and feedback.
2.  **Teacher Review**:
    - Sees Student Text + AI Feedback side-by-side.
    - Can accept, edit, or reject AI comments.
    - Adds personal encouraging note.
3.  **Result**: Student gets final grade + annotated text.
